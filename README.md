# Devx website Design 
##  DEVX is a modern developer education platform designed to help aspiring
    devs learn, build, and grow. The project includes responsive UI components,
    a mobile-first navigation system, a right-side sliding sidebar, and clean
    semantic HTML/CSS architecture.

## Features
    Fully responsive navigation bar
    Modern layout using custom variables (CSS vars)
    Scalable component-based structure
    Optimized assets and typography
## Tech Stack
     HTML5 (semantic structuring)
    CSS3 (Flexbox, Custom Properties, Media Queries)
    JS (for the nav and footer component)

## Key Features
    ✔ Responsive Navigation Bar

    Works seamlessly across mobile, tablet, and desktop using media queries.
    

    
    ✔ Scalable UI
    
    Built with reusable classes and CSS variables for colors, fonts, and spacing.
    
    ✔ Cross-browser Compatible
    
    Tested on Chrome, Firefox, Edge, and Safari.
